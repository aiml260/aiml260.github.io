Course Requirements and Evaluation:

● Homework
The homework is due every week in class on Sunday.

Problem sets will be reading papers or exploring existing code and control methods for soft robotics. They will be due (virtually) every week, on Sunday in class. Problems will be discussed in class, and often will be designed to investigate or re-prove results from research in this area. Problems might require coding or develop hardwares. For code, we recommend either Python (eg. using numpy and scipy), Matlab, or R. For hardware: using Joystick controls + designing electric boards for the control systems.

● Midterm Project
Details given in class, but this will reflect the final project in nature. If the final project is to be a continuation of the midterm project (which is expected), significant additional progress must be made.

● Final Project
The final project is intended to give students, in groups of 2-3, the opportunity to deep-dive into a specific area of interest in soft robotics, either code or hardware development. The detailed description and requirements will be put under the "Final Project" tab.
